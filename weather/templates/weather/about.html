About This App

This web application was created as a final project for the course INF601 - Advanced Programming in Python by Tanzeem Siddique. The goal of this project is to integrate real-world data using an API and generate useful suggestions for end users using the Django web framework.

The app allows users to enter any city name and fetch real-time weather information using the OpenWeatherMap API (accessed via RapidAPI). Based on the temperature and weather condition, it provides a personalized clothing recommendation.

Users can register or log in to save their searches and rate the clothing suggestions. A personal dashboard called My Ratings is available where users can review all their previous ratings and feedback.

The application is built using Django and styled with Bootstrap for responsive and user-friendly design. A Bootstrap modal is used to enhance interactivity when users rate their suggestions.

This project demonstrates the use of Django models, forms, views, templates, authentication, third-party APIs, and Bootstrap components to create a full-stack, data-driven web application.

Thank you for using this app!
